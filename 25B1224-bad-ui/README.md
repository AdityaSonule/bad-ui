# Volume Slider UI

A simple web UI demo showing a volume slider design built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main page markup
- `style.css` — layout and design styles
- `script.js` — interactive behavior for the slider

## Usage

1. Open `index.html` in a web browser.
2. Use the slider interface to change the volume value.

## Notes

- This is a static UI demo intended for quick local preview or GitHub hosting.
- No build step is required.
